# AnkCompiler
A CLI tool for compiling Anki decks, defined in Markdown.
## Installation
Run `pip install ankcompiler`
## Usage
Run `ankc --help` for usage information.
### Examples
An example source deck can be found in the [test deck](tests/decks/sample.md)
